# Cbit-Elearning-Website
Frontend UI project of CBIT E-Learning Platform built using HTML and CSS. This project focuses on layout and design. No backend functionality is implemented.

## Project Description
The project represents a simple UI design of a CBIT E-Learning Platform website.  
It includes:

- Navigation bar
- Hero section
- Login form UI
- Course cards section
- Footer section

 Note: This project focuses only on frontend design.  
Buttons and login functionality are not connected to any backend.

##  Technologies Used
- HTML5
- CSS3

##  How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
